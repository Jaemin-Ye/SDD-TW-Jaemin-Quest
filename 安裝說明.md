# 安裝與執行說明

## 前置需求

請確保已安裝 Python 3.11 或以上版本。

## 安裝步驟

1. 開啟終端機並切換到專案目錄：
```bash
cd "c:\Users\jaemin_ye.PFT\Python\AIxBDD\SDD.tw-Examples"
```

2. 安裝相依套件：
```bash
python -m pip install -r requirements.txt
```

或如果你使用的是 `py` 指令：
```bash
py -m pip install -r requirements.txt
```

## 執行 BDD 測試

執行所有測試（已標記 @skip 的會被略過）：
```bash
behave
```

或使用 Python 模組方式執行：
```bash
python -m behave
```

## 專案結構

```
.
├── src/                    # 原始碼目錄
│   ├── product.py         # Product 實體類別
│   ├── order_item.py      # OrderItem 實體類別
│   ├── order.py           # Order 實體類別
│   └── order_service.py   # OrderService 服務類別
├── features/              # BDD feature 檔案目錄
│   ├── order.feature      # 訂單優惠功能規格
│   ├── environment.py     # Behave 環境設定
│   └── steps/             # Step definitions
│       └── order_steps.py # 訂單步驟定義
├── requirements.txt       # Python 相依套件
└── behave.ini            # Behave 設定檔
```

## 手動測試

如果 behave 無法執行，可以使用手動測試腳本：
```bash
python test_manual.py
```
